Zelalem's Travel Guide

Your ultimate guide to exploring New Zealand!

Site Purpose

This site helps travelers plan their trips to New Zealand by providing information about top destinations, travel tips, itineraries, accommodations, and cultural insights.

Scenarios

  • As a tourist planning a 2-week trip, which destinations should I prioritize for a balance of nature and culture?
  • Where can I find reliable information about local accommodations and transport options?

Color Schema

  • Primary Color: #1E90FF (Dodger Blue) – used for headings, buttons, and important highlights.
  • Secondary Color: #F0F8FF (Alice Blue) – used for page backgrounds and content areas.
  • Accent Color: #FF4500 (Orange Red) – used for call-to-action highlights.

Typography

  • Headings: Montserrat – used for all headings (H1, H2, H3).
  • Body: Open Sans – used for paragraphs and descriptive text.
  • Special Sections: Roboto – used for navigation bar, footer, and buttons.

Wireframe

Desktop View: Header → Hero Image → 3-column content (Destinations, Travel Tips, Popular Activities) → Footer

Mobile View: Header → Hero Image → Single column stacked sections → Footer

CSS / Styling

The site uses an external CSS file named style.css. All selected colors and typography are applied according to the plan. The layout is responsive for desktop and mobile views.

Testing Plan

  • Validate HTML/CSS using W3C Validator
  • Check accessibility with WAVE or similar tools
  • Test color contrast for readability
  • Perform basic SEO checks: page titles, alt text, meta descriptions
  • Test responsiveness and performance on multiple devices
